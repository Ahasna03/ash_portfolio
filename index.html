Projects

my latest work

Heart Disease Prediction Using K-Nearest Neighbors (KNN)

Tools Used: R, Tidyverse, Caret, ggplot2

This project builds a KNN classification model to predict the presence of heart disease based on patient health data. It enables early intervention by using visualizations and model evaluation to interpret prediction accuracy.

ChildWell-Being Analysis Using Young Lives Data

Tools Used: Power BI, SQL Server, Report Builder

This project visualizes the well-being of children in Ethiopia and India using health, education, and household indicators from the Young Lives dataset. Dashboards enable stakeholders to understand and address poverty-related issues effectively.

PoliGEN Document Summarizer and Policy Generator Web App

Tools Used: Flask, Gemini AI, Python, HTML, CSS, JavaScript

This web application uses Flask and Gemini AI to automatically summarize policy documents, enhancing efficiency in policy analysis. Users can upload documents and receive concise summaries, with AI models providing high-quality, context-aware outputs.

Greater Manchester Domestic Energy Performance Analyzer

Tools Used: Microsoft Azure, Power BI, SQL Server

This project examines domestic energy performance trends in Greater Manchester using EPC data over a 16-year period. It supports energy policy by identifying key patterns and geographic hotspots of poor energy efficiency.

Comprehensive Power BI Dashboard on the Modern Summer Olympics (2000–2016)

Tools Used: Power BI, SQL

This dashboard provides a complete visual overview of Olympic participation, medal trends, and gender equality between 2000 and 2016. It enables exploration of athlete performance and national progress in a dynamic format.

Pokémon Performance Analysis Using K-Means Clustering

Tools Used: R, Cluster, Factoextra, Power BI

This clustering project analyzes Pokémon attributes like Speed, Attack, and Defense to group them by performance. It visualizes patterns in Pokémon types and supports data-driven strategic choices in games.

Project Management App

Tools Used: Power Apps, Power Automate, Dataverse

This app helps streamline research project workflows with task tracking, team collaboration, and milestone visualization. It's built using Power Platform tools for ease of use and seamless integration with Microsoft services.

Crime Hotspot Prediction and Pattern Analysis

Tools Used: Python, Scikit-learn, Pandas, Power BI

This project uses crime data to identify spatial and temporal hotspots using machine learning. It supports proactive policing and resource allocation through predictive analytics and interactive visualizations.

Association Rules Mining on Market Basket Dataset

Tools Used: R, Power BI

Applied Association Rules Mining on a Market Basket dataset using the \verb|apriori()| algorithm, visualizing insights into customer purchasing behavior in a Power BI dashboard.
